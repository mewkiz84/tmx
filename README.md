tmx
===

This package provides support for reading [tmx][] (Tile Map XML) files, used by
the map editor [Tiled][].

[tmx]: https://github.com/bjorn/tiled/wiki/TMX-Map-Format
[Tiled]: https://github.com/bjorn/tiled/
